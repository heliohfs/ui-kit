# Text Editor

To view `EditText` component and it's props, check out [@tecsinapse/text-editor](https://github.com/tecsinapse/text-editor).
After importing `@tecsinapse/ui-kit` and `@tecsinapse/text-editor`, all you have to do is:

```
import { ThemeProvider } from '@tecsinapse/ui-kit';
import { EditText } from '@tecsinapse/text-editor';

const App = () => {
  return (
    <ThemeProvider variant="orange">
      <EditText />
    </ThemeProvider>
  );
};
```

It's necessary to specify the prop `skinUrl` of component to point the location of the editor stylesheet.
Skins can be found on folder `node-modules/tinymce/skins/ui` and copied to public asset folder of your application.
You can check our [github](https://github.com/tecsinapse/text-editor/tree/master/storybook-public/oxide) to view how the folder is mounted according to code snippet below.
