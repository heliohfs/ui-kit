# Uploader

To view `Uploader` component and it's props, check out [@tecsinapse/uploader](https://github.com/tecsinapse/uploader).
After importing `@tecsinapse/ui-kit` and `@tecsinapse/uploader`, all you have to do is:

```
import { ThemeProvider } from '@tecsinapse/ui-kit';
import { Uploader } from '@tecsinapse/uploader';

const App = () => {
  return (
    <ThemeProvider variant="orange">
      <Uploader />
    </ThemeProvider>
  );
};
```
