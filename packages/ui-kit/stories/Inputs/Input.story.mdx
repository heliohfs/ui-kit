# Input

The `Input` component can receive the following props:

<Props of={Input} />

You can customize you own mask. To place a Date mask for example:
```
import { Input } from '@tecsinapse/ui-kit';

const App = () => {
  const DATE_MASK = [/\d/, /\d/, '/', /\d/, /\d/, '/', /\d/, /\d/, /\d/, /\d/];
  return <Input name="date" label="Input" mask={DATE_MASK} />;
};
```

Check our implementation on [github](https://github.com/tecsinapse/ui-kit/tree/master/src/Inputs).
