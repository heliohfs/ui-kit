# Login

To view the components and it's props, check out [@tecsinapse/login](https://github.com/tecsinapse/login).
After importing `@tecsinapse/ui-kit` and `@tecsinapse/login`, all you have to do is:

```
import { ThemeProvider } from '@tecsinapse/ui-kit';
import { Login } from '@tecsinapse/login';

const App = () => {
  return (
    <ThemeProvider variant="orange">
      <Login />
    </ThemeProvider>
  );
};
```
