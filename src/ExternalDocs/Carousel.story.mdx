# Carousel

A Carousel is composed by Slides. To view `AutoRotatingCarousel` and `Slide` components and it's props, check out [@tecsinapse/carousel](https://github.com/tecsinapse/carousel).
After importing `@tecsinapse/ui-kit` and `@tecsinapse/carousel`, all you have to do is:

```
import { ThemeProvider } from '@tecsinapse/ui-kit';
import { AutoRotatingCarousel, Slide } from '@tecsinapse/carousel';

const App = () => {
  return (
    <ThemeProvider variant="orange">
      <AutoRotatingCarousel>
        <Slide />
      </AutoRotatingCarousel>
    </ThemeProvider>
  );
};
```
