# Chat

To view chat component and it's props, check out [@tecsinapse/chat](https://github.com/tecsinapse/chat).
After importing `@tecsinapse/ui-kit` and `@tecsinapse/chat`, all you have to do is:

```
import { ThemeProvider } from '@tecsinapse/ui-kit';
import { Chat } from '@tecsinapse/chat';

const App = () => {
  return (
    <ThemeProvider variant="orange">
      <Chat />
    </ThemeProvider>
  );
};
```
